# odin-recipes

This is my first project from **The Odin Project Foundations curriculum**.

## About the project

Odin Recipes is a simple multi-page website built using only **semantic HTML and CSS**.  
The goal of this project was to practice structuring web pages and working with basic layout techniques.

## What I learned

During this project I practiced and improved:

- Semantic HTML (header, main, section, figure, etc.)
- CSS layout fundamentals (Flexbox & Grid)
- Basic responsive layout techniques
- BEM methodology for class naming
- Image handling and aspect ratios

## Technologies used

- HTML5
- CSS3

## Notes

Most of the concepts were already familiar to me, but this project helped me better understand:
- how to structure real-world pages
- how to organize CSS in a scalable way
- how to think in components using BEM
